# 雪豹助手

一个自己用来记录资产和收支的小工具。  
目前主要做了资产记录、收支记录、三角洲资产预留入口，以及一个用于下载 App 的网页。

## 主要功能

- 记录房地产、电子产品、饰品、基金、游戏资产等不同类型的资产。
- 房地产会显示面积、所在地、区域、详细位置等字段。
- 电子产品、饰品、基金、游戏资产不会要求填写房地产相关信息。
- 支持记录购入价、当前估值、购入日期和备注。
- 支持自定义资产类别。
- 收支模块可以记录收入和支出，并查看本月收入、本月支出和结余。
- 预留了三角洲资产模块，后面可以继续接登录和游戏资产查看。
- 数据目前保存在手机本机。

## 当前版本

- App：`v3.5.2`
- 安装包：`雪豹助手-v3.5.2.apk`
- 网站发布包：`雪豹助手-website-official.zip`

## 下载网站

`website/` 目录是下载页源码。  
部署后可以通过这些地址检查资源：

- `/check.html`
- `/app-icon.png`
- `/xuebao-assistant.apk`

如果手机浏览器不能下载 APK，建议换系统浏览器、Chrome 或 Edge，微信和 QQ 内置浏览器可能会拦截安装包。
