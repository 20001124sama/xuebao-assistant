# 雪豹助手

这是我自己做的一个个人资产记录 App，目前主要用来记一些日常资产和收支。  
一开始是想做一个类似“有数”的资产管理工具，后来慢慢加了房地产、电子产品、饰品、基金、游戏资产、收支记录和下载网站这些东西。

当前版本是 `v3.5.2`。

## 现在能做什么

- 记录不同类型的资产，比如房地产、电子产品、饰品、基金、游戏资产等。
- 房地产会单独显示面积、所在地、区域、详细位置这类字段。
- 电子产品、饰品、基金、游戏资产这些类型不会强制填写房地产字段。
- 每个资产可以记录购入价、当前估值、购入日期、备注等信息。
- 第二个模块现在是“收支”，可以记收入，也可以记支出。
- 收支会区分 `+` 和 `-`，方便看本月收入、支出和结余。
- 预留了“三角洲资产”模块，后面可以继续接微信或 QQ 登录，再查看游戏内资产。
- App 进入时有三秒启动动画，用的是当前项目里的图标。
- 数据目前保存在手机本机，适合个人记录使用。

## 这次 v3.5.2 改了什么

- 把原来的“开支”模块改成了“收支”模块。
- 新增收入记录，不再只能记支出。
- 调整了收支统计，现在能看本月收入、本月支出和结余。
- 修了网站下载页的资源路径，把图标和 APK 放到了网站根目录，减少部署后找不到文件的问题。
- 加了 `check.html`，部署后可以直接检查图标和安装包有没有正常上传。
- 给 Netlify 加了 `_headers`，让 APK 下载时按 Android 安装包处理。

## 安装包

如果只是想安装 App，可以直接下载：

- `雪豹助手-v3.5.2.apk`
- `snow-leopard-assistant-v3.5.2.apk`

手机安装时如果提示风险或拦截，需要在系统里允许“安装未知来源应用”。  
微信、QQ 内置浏览器经常会拦截 APK，建议用手机自带浏览器、Chrome 或 Edge 下载。

## 下载网站

网站文件在 `website/` 目录里。  
我另外打了一个可以直接上传的包：

- `雪豹助手-website-official.zip`

部署到 Netlify 后可以测试这几个地址：

- `/`：下载首页
- `/check.html`：资源检测页
- `/app-icon.png`：网站图标
- `/xuebao-assistant.apk`：APK 下载直链

如果首页图标不显示，先打开 `/check.html` 看资源有没有正常部署。  
如果 `/app-icon.png` 或 `/xuebao-assistant.apk` 打不开，通常就是部署包没传对，或者 Netlify 还在显示旧部署。

## 本地构建

Android 项目在 `app/` 目录。  
构建需要 JDK、Gradle 和 Android SDK。

这个 App 本质上是 Android WebView 加载本地网页资源，所以核心功能不依赖服务器。网站主要是用来展示和下载安装包。
